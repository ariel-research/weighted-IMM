# חלוקת קורסים הוגנת 
 **אלגוריתם איטראטיבי למציאת שידוך מקסימלי**
## דרישות
python, pip

## התקנה
צרו סביבה וירטואלית:
```
python -m venv venv
```

הפעילו אותה:
```
source venv/bin/activate
```

התקינו את fairpyx:

```
pip install fairpyx
```


## הדגמת שימוש באלגוריתם

הריצו את הקובץ example.py:
```
python example.py
```
