# חלוקת קורסים הוגנת 
 **אלגוריתם איטראטיבי למציאת שידוך מקסימלי**
## דרישות
`python 3.10+`, `pip`.

## התקנה והרצה
```
git clone https://github.com/ariel-research/weighted-IMM
```

צרו סביבה וירטואלית:
```
cd weighted-IMM
python -m venv venv
```

הפעילו אותה: בלינוקס - 
```
source venv/bin/activate
```
בחלונות - 
```
venv/Scripts/activate.bat
```

התקינו את fairpyx:
```
pip install fairpyx
```


## הדגמת שימוש באלגוריתם

הריצו את הקובץ example.py:
```
python example.py
```
